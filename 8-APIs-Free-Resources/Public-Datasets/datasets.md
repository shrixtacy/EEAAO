# Public Datasets

## Government and Institutional Data

### Data.gov (United States)
- **What It Provides**: US government datasets across all agencies and topics
- **Access**: Free, no registration required for most datasets
- **Formats**: CSV, JSON, XML, various formats
- **Documentation Quality**: Varies by dataset, generally good metadata
- **Update Frequency**: Varies by source agency
- **Use Cases**: Policy research, civic apps, data journalism, academic projects

### European Data Portal
- **What It Provides**: European Union open data across member countries
- **Access**: Free, no registration required
- **Formats**: Multiple formats including CSV, RDF, JSON
- **Documentation Quality**: Good with multilingual support
- **Update Frequency**: Regular updates from member states
- **Use Cases**: European market analysis, policy research, comparative studies

### World Bank Open Data
- **What It Provides**: Global development data, economic indicators, demographics
- **Access**: Free, API available
- **Formats**: CSV, Excel, XML, JSON via API
- **Documentation Quality**: Excellent with detailed methodology
- **Update Frequency**: Annual and quarterly updates
- **Use Cases**: Economic analysis, development research, global comparisons

## Scientific and Research Data

### NASA Open Data
- **What It Provides**: Space, earth science, climate, and astronomical data
- **Access**: Free, some datasets require registration
- **Formats**: Various scientific formats, CSV, NetCDF, HDF
- **Documentation Quality**: Excellent with scientific metadata
- **Update Frequency**: Continuous for satellite data, periodic for research
- **Use Cases**: Climate analysis, space research, educational projects

### Kaggle Datasets
- **What It Provides**: Community-contributed datasets for machine learning
- **Access**: Free with Kaggle account
- **Formats**: Primarily CSV, some JSON and images
- **Documentation Quality**: Varies, community-driven descriptions
- **Update Frequency**: Static datasets, new ones added regularly
- **Use Cases**: Machine learning practice, data science competitions, research

### Google Dataset Search
- **What It Provides**: Search engine for datasets across the web
- **Access**: Free search, dataset access varies
- **Formats**: All formats, depends on source
- **Documentation Quality**: Aggregated from various sources
- **Update Frequency**: Continuous indexing
- **Use Cases**: Dataset discovery, research, academic projects

## Dataset Selection Guidelines

### For Beginners
- **Start with**: Kaggle datasets, JSONPlaceholder, simple CSV files
- **Focus on**: Clean, well-documented datasets
- **Avoid**: Complex scientific formats until you need them

### For Machine Learning
- **Consider**: Kaggle, UCI ML Repository, Google Dataset Search
- **Look for**: Labeled data, appropriate size, clear features
- **Check**: Data quality, missing values, bias considerations

### For Research
- **Use**: Government sources, academic repositories, institutional data
- **Verify**: Data provenance, methodology, update frequency
- **Cite**: Properly attribute data sources in publications

### Data Quality Checklist
- **Completeness**: Are there missing values or gaps?
- **Accuracy**: Is the data from a reliable source?
- **Timeliness**: Is the data current enough for your needs?
- **Relevance**: Does the data match your research questions?
- **Format**: Can you easily work with the data format?

### Legal and Ethical Considerations
- **Licensing**: Check usage rights and attribution requirements
- **Privacy**: Ensure personal data is properly anonymized
- **Terms of Use**: Understand restrictions on commercial use
- **Attribution**: Properly credit data sources
- **Bias**: Consider potential biases in data collection
