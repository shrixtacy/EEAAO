# Design Tools

## UI/UX Design Software

### Figma
- **Problem It Solves**: Collaborative interface design and prototyping
- **When NOT to Use**: Complex illustration work, print design
- **Free Tier**: 3 Figma files, unlimited personal drafts
- **Alternatives**: Adobe XD (Adobe ecosystem), Sketch (Mac-only)
- **Learning Curve**: Low to moderate - intuitive interface

### Adobe XD
- **Problem It Solves**: UI/UX design with Adobe Creative Cloud integration
- **When NOT to Use**: When team collaboration is primary need
- **Free Tier**: 1 shared document, limited cloud storage
- **Alternatives**: Figma (better collaboration), Sketch (Mac-only)
- **Learning Curve**: Moderate - Adobe interface patterns

## Icon and Illustration Tools

### Heroicons
- **Problem It Solves**: Consistent, high-quality SVG icons for web projects
- **When NOT to Use**: Unique brand identity needs, complex illustrations
- **Free Tier**: Completely free and open source
- **Alternatives**: Feather Icons, Lucide, Font Awesome
- **Learning Curve**: None - copy and paste SVG code

### Undraw
- **Problem It Solves**: Consistent illustration style for web projects
- **When NOT to Use**: Unique brand requirements, specific industry imagery
- **Free Tier**: All illustrations free with attribution
- **Alternatives**: Storyset, Drawkit, Humaaans
- **Learning Curve**: None - download and use

## Color and Typography

### Coolors
- **Problem It Solves**: Color palette generation and exploration
- **When NOT to Use**: Brand-specific color requirements
- **Free Tier**: Generate unlimited palettes, save 10 palettes
- **Alternatives**: Adobe Color, Paletton, Material Design Colors
- **Learning Curve**: None - click to generate

### Google Fonts
- **Problem It Solves**: Web-safe typography with easy implementation
- **When NOT to Use**: Unique brand typography, print design
- **Free Tier**: All fonts completely free
- **Alternatives**: Adobe Fonts (subscription), Font Squirrel (free)
- **Learning Curve**: None - copy CSS link

## Prototyping Tools

### Framer
- **Problem It Solves**: High-fidelity interactive prototypes
- **When NOT to Use**: Simple static mockups, when code is faster
- **Free Tier**: 3 projects, basic features
- **Alternatives**: Principle, ProtoPie, InVision
- **Learning Curve**: Moderate to high - animation concepts required

## Design Resources

### Unsplash
- **Problem It Solves**: High-quality stock photography
- **When NOT to Use**: Specific product shots, brand-specific imagery
- **Free Tier**: Unlimited downloads with attribution
- **Alternatives**: Pexels, Pixabay, Burst by Shopify
- **Learning Curve**: None - search and download

## Tool Selection Guidelines

### For Beginners
- **Start with**: Figma, Heroicons, Google Fonts, Coolors
- **Focus on**: Learning design principles over tool mastery
- **Avoid**: Complex illustration tools until you need them

### For Developers
- **Prioritize**: Tools that export clean code and assets
- **Choose**: Web-focused tools over print design software
- **Consider**: Integration with your development workflow

### For Teams
- **Collaboration First**: Figma for real-time collaboration
- **Consistency**: Shared libraries and design systems
- **Handoff**: Tools that generate specs for developers
